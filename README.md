Parsclick
==================

Parsclick is the unique and best Persian online training library.
This ia made by:

- Vanilla PHP
- Vanilla JavaScript
- jQuery and Ajax
- Composer
- Elixir
- Node
- YouTube API
- Using FileSystem
- Using OOP pattern
- No framework which makes it very fast
